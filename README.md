# Curso: Sua PRIMEIRA aplicação WEB com DJANGO por Pythonando 

---

## 📌 Sobre o Projeto
O objetivo principal foi entender o fluxo de funcionamento do Django (MVT) e como o framework lida com requisições, banco de dados
e renderização de páginas.

### O que foi implementado:

* Estruturação de um projeto e app Django.
* Criação de modelos (Models) para o banco de dados.
* Manipulação de dados através do Django Admin.
* Criação de Views e Templates para exibição de conteúdo.

## 🛠️ Tecnologias Utilizadas
* **Python 3.12.8**
* **Django** (Framework Web)
* **SQLite** (Banco de dados padrão do Django)
* **Markdown** (Para anotações e documentação)

  ## Como executar o projeto:

1. **Clone o repositório:**
   
   ```bash
   git clone https://github.com/santos-caroline/testeDjango.git
   cd seu-repositorio
   
3. Instale as dependências:
   
````bash
pip install django
`````

3. Execute as migrações (Banco de Dados):
   
````bash
python manage.py makemigrations
python manage.py migrate
````

4.Inicie o servidor:

````bash
python manage.py runserver
````

Acesse: ````http://127.0.0.1:8000/````
